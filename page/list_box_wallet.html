• No single point of failure

    개인키가 단일 지점에 존재하지 않으므로 해킹으로부터 안전하고 암호화된 프로토콜을 통해 개인키 복원 없이 디지털 서명을 생성
  • Full-fledged Web3 Wallet

    Uniswap, Opensea 등 다양한 Dapp 서비스와 완벽하게 호환되는 Web3 지갑
  • No mnemonic backup

    사용자가 mnemonic을 관리할 필요가 없이 간편한 신원 인증을 통해 계정 복구
  • Self custodial wallet

    사용자는 자신의 자산에 대한 완전한 통제권을 행사
  • Multi devices with ONE address

    MPC 기술을 통해 하나의 주소로 안전하게 다중 기기 이용이 가능
  • Multi platforms supported

    Browser Extension, Mobile (iOS/Android), In-browser Wallet
